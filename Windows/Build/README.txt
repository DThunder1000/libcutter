Windows libcutter plugin for Sure Cuts A Lot 4

Install Instructions:

1. Copy libLibCutter.xml and libLibCutterPlugin.dylib into:
/Users/{YOUR USERNAME HERE}/Documents/SCALCutPlugins

2. Start Sure Cuts A Lot 4 and go to the Cutter menu and choose Manage Cutters and you should be able to add Cricut as an option now.

3. In order for the communication to work properly, you will need a valid key file named keys.h and placed in the SCALCutPlugins folder. The proper key file is not distributed with this project. For more information on this:
http://jimlund.org/build-to-spec/Setting%20Up%20Libcutter%20-%20Cricut%20Hacking%20Wiki.html
